# Bedrock

Um laboratório de **vibe coding** onde a IA é direcionada por um arquiteto de software experiente.

## Propósito

Este repositório é um experimento real de desenvolvimento assistido por IA, validando um cenário onde:

- **Humano define O QUÊ e PORQUÊ** — Requisitos, decisões arquiteturais, direção estratégica
- **IA executa o COMO** — Implementação, código, testes, refatorações

## Abordagem

O objetivo é desenvolver software direcionando code agents com:

- **Limites claros** — Escopo bem definido para cada tarefa
- **Guardrails** — Convenções, padrões e restrições que a IA deve seguir
- **Supervisão arquitetural** — Revisão humana das decisões técnicas importantes

## Stack

- .NET 10.0
- C# (latest)
