Decisoes arquiteturais sobre padroes especificos do PostgreSQL e Npgsql.

## ADRs

| ADR | Titulo | Status | Rule |

## Escopo

- **PG-***: Regras que definem padroes obrigatorios para uso do Npgsql, protocolo COPY binario e configuracao de conexoes PostgreSQL.
